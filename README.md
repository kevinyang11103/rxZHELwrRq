# Python高校迎新管理系统 python798

## 项目概述

本项目是基于Python的高效迎新管理系统，旨在为高校迎新活动提供便捷的管理与操作平台。

## 技术栈

- 后端：Python + Django
- 前端：Vue
- 数据库：MySQL

## 系统角色

- 管理员
- 学生

## 运行环境

- 后端：Python 环境推荐 Python 3.x，Django 版本依项目具体要求而定。
- 前端：若原电脑 Node.js 版本高于 14，可配置 cnpm 环境。
- 数据库：MySQL 推荐版本 5.x 或以上。

## 功能模块

系统功能模块包括但不限于：

- 注册与登录
- 学生信息管理
- 迎新活动管理
- 数据统计与分析
